# Service Fabric CLI (sfctl)

Command-line interface for interacting with Azure Service Fabric clusters and
their related entities.

## Installation

```bash
pip install sfctl
```

### Launching

```bash
sfctl <command>
```

For more information, specify the `-h` flag:

```bash
sfctl -h
```
